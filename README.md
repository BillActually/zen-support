  ## FAQ                                                                                                                                                                                                       
                                                                                                                                                                                                               
  **What audio formats are supported?**                                                                                                                                                                        
  MP3, M4A, WAV, and other formats supported by iOS.                                                                                                                                                           
                                                                                                                                                                                                               
  **How do I add music?**                                                                                                                                                                                      
  There are two methods of importing music:
    Method 1 - Import music through iTunes File Transfer
      1. plug your phone into a computer
      2. open iTunes
      3. select your phone
      4. select File Sharing
      5. find Zen in the list of apps and select it
      6. drag-and-drop your music files
      7. eject your phone
      8. the songs are now on the app
    
  Method 2 - Wireless file sharing via the app
      1. open the Zen Music Player on your phone
      2. tap the gear icon in the top-left corner of the Library Tab
      3. tap Start/stop server
      4. start the server
      5. on a computer that has your music files, open an internet browser and go to the URL specified on the app's File Sharing page
      6. once on the server browser URL page, select Upload in the left-hand menu
      7. drag and drop music or click Choose Files to select your music library
      8. wait for the transfer to finish
      9. the songs are now on the app

  **How do I add music?**

   Using Method 2 described above, upload playlists as .m3u/.m3u8 files into the server browser URL page after your music is already installed.
                                                                                                                                                                                                               
  **How do I use the widget?**                                                                                                                                                                                 
  Long-press on your home screen, tap +, search for "Zen", and add the widget.
